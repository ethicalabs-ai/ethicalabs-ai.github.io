BlossomTune Orchestrator

BlossomTuneLLM

BlossomTuneLLM is an evolution of the FlowerTune LLM benchmarking initiative by Flower Labs. It showcases our ResearchOps expertise in automating complex federated fine-tuning simulations. We've built this by providing a cookiecutter template for rapid setup and later by leveraging Docker containers for flexible deployments across cloud, edge devices, and laptops.

BlossomTuneLLM-MLX

A native implementation of our federated fine-tuning initiative for Apple Silicon.

By leveraging the MLX-LM framework, this project enables on-device federated learning on MacOS, reducing reliance on expensive cloud GPUs and championing privacy and sustainability.

A synthetic data generation tool for LLMs that addresses data scarcity and privacy concerns. This open-source project automates the creation of diverse, high-quality datasets for fine-tuning Language Models in an ethical and privacy-preserving manner, without relying on sensitive real-world data.
